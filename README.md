# ROP 汇编器

## 简介

汇编器(asm.py) python编写完成，可以极大程度简化ROP编写过程

## 用法

### Windows
```
python .exe .\asm.py xxx
```
例子:
```
python.exe .\asm.py .\Examples\Single-Line-Print.asm
```
### Linux
```
python ./asm.py xxx
```
例子:
```
python ./asm.py ./Examples/Single-Line-Print.asm
```
